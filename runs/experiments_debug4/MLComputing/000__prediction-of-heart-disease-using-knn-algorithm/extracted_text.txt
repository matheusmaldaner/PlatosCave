Algorithm Introduction: 
The K-nearest neighbors(KNN) algorithm is a 
machine learning method used for predictions, 
including heart disease prediction . It is a 
supervised learning algorithm which works on the 
principle of similarity. 
In context to the heart disease prediction, the KNN 
algorithm uses a labelled dataset that involves 
information of various attributes such as age, 
gender, cholesterol levels, blood pressure and other 
factors. Each individual in the given dataset is 
labelled as either they are suffering with the heart 
disease or not suffering with any heart disease. 
When an individual's data is provided, the KNN 
algorithm calculates the similarity between the 
individual and the labelled data points in the 
dataset. It does it by measuring the distance 
between the individual's attribute values of the 
labelled data points. 
The KNN algorithm then selects the K-nearest 
neighbors(data points with the closest attribute 
values) and determines the majority class among 
these neighbors. This majority class is used to 
predict if the individual is expected to be diagnosed 
with the heart disease or not. 
Its very important to note the choice of K(number 
of neighbors) which can have an effect on the 
algorithm's performance. Additionally, feature 
selection and data preprocessing are the crucial 
steps in preparing the dataset for accurate 
predictions. Overall, the KNN algorithm can be 
considered as one of the useful tools for detecting 
the heart diseases depending on the similarities 
between the individuals and their attributes. 
However, its always recommended to consult the 
medical professionals for accurate diagnosis and 
treatment.

III. PROPOSED METHODOLOGY 
Problems with regression and classification are 
addressed by the supervised machine learning 
technique known as the K-Nearest Neighbors 
(KNN) algorithm. Joseph Hodges and Evelyn Fix 
proposed this algorithm in 1951, which was 
subsequently developed by Thomas Cover. 
The KNN method is a popular and adaptable 
machine learning technique that is mostly used 
due to its simplicity and ease of implementation. 
No presumptions on the underlying data 
distribution are necessary. It is an adaptable 
option for a variety of dataset types in regression 
and classification applications because it can 
handle both numerical and categorical data. This 
approach, which is non-parametric, bases its 
predictions on how similar the data points are to 
one another across datasets. The sensitivity of K- 
NN to outliers is lower than that of other methods. 
Using a distance metric, such as Euclidean 
distance, the K-NN algorithm locates the K- 
nearest neighbors (KNN) of a given data point. 
The majority vote or average of the K neighbors 
then controls the class or value of data points. 
Using the local structure of the data, this method 
enables the KNN algorithm to adjust to various 
patterns and generate predictions. 
The basic, non-parametric algorithm that doesn't 
make any assumptions about the distribution of 
the data is K-Nearest Neighbor (KNN). This 
approach is based on the Euclidean distance 
theory, which states that examples with 
comparable characteristics tend to appear close 
together within a dataset. In the event that each 
instance has a classification tag applied, the 
closest neighbor's class can be used to determine 
the tag value of an unclassified instance. The 
current modelling entails performing every step 
thrice (repeats = 3), with a numerical value for K 
of 10 (number = 10) and averaging each of the 
three trials. 
Certainly, determining coronary artery disease 
(CAD) using the KNN is an interesting 
application of machine learning. Let‚Äôs delve into 
it: 
K-Nearest Neighbors (KNN): 
Regression and classification tasks are the two 
uses for the supervised learning algorithm KNN. 
That built on the idea that the similar data points 
tend to have similar labels. KNN, a non- 
parametric method, meaning it doesn‚Äôt assume 
any kind of specific functional form for the data. 
Unlike some other algorithms, KNN memorizes 
the training set during training and performs most 
of its computation during classification. 
The key steps in KNN are: 
ÔÇ∑ 
Choose a value for K. 
ÔÇ∑ 
Determine 
the 
distance 
(e.g., 
Euclidean distance) between the new 
data point and all the prior data points. 
ÔÇ∑ 
Select the K-nearest neighbors based 
on small distances. 
ÔÇ∑ 
Assigns new data point to every class 
that appears most commonly amongst 
the K neighbor values. 
Working of KNN Algorithm: 
Given a new data point, KNN: 
1. Finds the K-nearest neighbors based on 
distance. 
2. Determines the majority class among 
those neighbors. 
3.  Gives that class ownership of the new 
data piece.

o Odd values of K are preferred to 
avoid ties during voting.

introduce noise.
